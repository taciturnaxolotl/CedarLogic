Saving/Opening A Circuit

1. To save circuit, simply select save from the file-menu. A standard Windows save dialog box will appear.

2. To open a circuit, select open from the file menu. A standard Windows open dialog box will appear. Select the appropriate directory in which you saved the file you wish to open.

Note that CEDAR Logic can only have one circuit file open at a time.

3. For this tutorial, save the circuit. You will need it in the next module.
