Shift Register

Pin(s)

Function

Output

Shows the result of the last shift operation.

Input

Bus which provides the number that may be loaded into the register.

SO (Shift Out)

Outputs the current value of the MSB.

SI (Shift IN)

Inputs a value to put into the LSB of the Shift.

R (Reset)

When high, resets the value in the register to 0. When low, does nothing.

L (Load)

When high, the input is loaded into the shift register.

<-

If shift is enabled, when high, the shift register will shift left. If shift is enabled, when low, the shift register will shift right.

S (Shift Enable)

When high, the shift register will perform a shift on the rising edge of the clock. When low, the shift register just holds its current value.

Note: Load and Reset can be asynchronous or synchronous by double clicking on the gate and choosing the appropriate dialog.
