Adding Text

1. Selected the TEXT object from the gate palette

2. Drag the TEXT object to the appropriate position.

3. Double click on the TEXT object so that a dialog box appears.

4. Type Input in the LABEL portion of the dialog box and press enter. Note that labels can be placed anywhere on a circuit that you would wish.

5. To delete a label, simply select it like you would a gate and press the delete key. Play around with your test circuit by adding as many text labels as you want until you feel comfortable with the process. Once you are reasonably comfortable with it, delete the labels you added to the circuit and continue the tutorial.
