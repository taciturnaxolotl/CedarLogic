Undo/Redo

1. The Undo function of the program can be used to undo previous commands in the circuit. The Undo function can be performed by pressing CTRL-Z, using the undo shortcut on the toolbar, or selecting undo from the edit menu.

2. To review, delete the OR gate you added to the circuit earlier and save the file under a different name. When you are doing designs, it is important to save early and often so that you do not loose work or in case you need to revert to a previous design. Your circuit should look like the one below before you move on.

3. Select the portion of the partial full adder circuit as shown.

4. Delete the selected portion of the circuit.

5. Press CTRL-Z to undo the delete function to restore the circuit back to its original state.

6. The Redo function of the program can be used to redo previous commands in the circuit. The Redo function can be performed by pressing CTRL-Y, using the redo shortcut on the toolbar, or selecting redo from the edit menu.

7. Press CTRL-Y to redo the delete you just performed on the circuit.

8. The undo/redo functionality of the program can be used in almost all functional aspects of the program. The undo/redo function allows you to do an infinite number of undo/redo commands to your program.
