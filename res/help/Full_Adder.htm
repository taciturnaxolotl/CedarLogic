Full Adder

Pin(s)

Function

X Input

Number to be added.

Y Input

Number to be added.

CO (Carry Out)

Equals 1 whenever the sum exceeds 15.

CI (Carry IN)

This bit is added to the least significant bit of X and Y.

V (Overflow)

When high, a 2's complement overflow occurred. When low, an overflow did not occur. This bit is the XOR or the carry_in and carry_out of the most significant bit (sign bit).

S (Sum)

Outputs the sum of the last addition operation.
