TO/FROM Gates

1. TO/FROM gates are bidirectional junctions between wires. Each TO/FROM gate with the same name act like they are the same wire.

2. To edit the connection of a TO/FROM gate simply double click on the gate and a dialog box will appear.

3. Type in the appropriate name to form the desired connection and press enter. The text will change on the TO/FROM gate so that it reflects the text typed in. This indicates that the TO/FROM gate makes a connection to any of the other TO/FROM gates with the same name.
