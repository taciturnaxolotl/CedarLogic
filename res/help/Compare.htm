Compare

Function:

Performs an unsigned integer compare of the two inputs X and Y. These devices can be cascaded to compare numbers greater than 4 bits.

Pin(s)

Function

X Input

Number to be compared.

Y Input

Number to be compared.

Outputs

Reports results of compare operation.

Inputs

These bits are the carry_ins of previous comparisons.

>

Is high when X > Y.

=

Is high when X = Y.

<

Is high when x < Y.
