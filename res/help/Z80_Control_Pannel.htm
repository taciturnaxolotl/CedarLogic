Z80 Control Panel

The Z80 Control panel allows you to control the execution of Z80 programs as emulated by the Z80 gate.  The Z80 control panel is accessed by double clicking on the Z80 gate that you wish to control.

#

Description

1

Displays the primary registers and Interrupt mode.  The PC stands for program counter, and the SP stands for stack pointer.

2

Displays the alternate registers.

3

Provides three levels of executing code to debug programs:
Continuous: will run your program without stopping.

Instruction: will run one compleat instruction and then break.

T-State: will run until the next rising or falling edge of the click and then will break.

4

This keypad  emulates the keys on the ZAD trainer board.  The are only accessible from code when the ZAD mode is on or in smart mode. See section 6.  A key press is obtained by calling the ZAD operating system call KBRD at 0x0FEB or the ZAD system call SCAN at 0x0FEE.

See ZAD System calls.

5

This window will display the currently executing instruction.  It also displays a description of that instruction.

6

This selects the currently running ZAD mode.
off: disables any ZAD operating system commands.

on: forces ZAD operating system emulation despite memory conditions.

smart: only emulates the ZAD operating system if the targeted memory address contains a NOP.  This is default.

See ZAD System calls.

7

Emulates the ZAD trainer board's hexadecimal display.  This is used by calling HEXTO7 at 0x0FF1 followed by calling KBRD or SCAN.

See ZAD System calls.

8

This shows a pop-up which allows direct manipulation of the Z-80 registers.
