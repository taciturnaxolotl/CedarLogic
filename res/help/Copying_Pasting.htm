Copying/Pasting

1. Copying a selection can be performed by pressing CTRL-C, using the copy shortcut on the toolbar, or selecting copy from the edit menu.

2. Select the portion of the full-adder circuit as shown.

3. Copy the selection by pressing CTRL-C.

4. Pasting a selection can be performed by pressing CTRL-V, using the paste shortcut on the toolbar, or selecting paste from the edit menu.

5. Paste the selection by pressing CTRL-V into the circuit. Another copy of the selection appears on the canvas where your mouse pointer was when you pasted it.

The copied selection will follow the mouse pointer around until you left click on the gate canvas.

Note that since Cedars can only have one file open at a time, if you wish to insert one file into another file the best way to do this is to copy and paste between separate instances of the application.
