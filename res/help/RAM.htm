RAM

Pin(s)

Function

Address

Supplies the RAM address on which the operation is to be performed

Input

Supplies the input data to write to the RAM

Output

Supples the output data from the current operation of the RAM

W (Write Enable)

When high, the data placed on the input lines will be written to the current address on the next rising edge of the clock. The output is placed in a high impedance state during a write. Note that the input can be wired to the output to form a bidirectional data bus. When low, the RAM will perform a read operation by placing the contents stored at the current location on the output bus.

OE (Output Enable)

When high, the data of stored in the current address specified by the address pins will be outputted. When low, the value will be high impedance.

Note: To load or save RAM files, double click on the RAM.
