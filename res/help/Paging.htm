Paging

1. Delete the clock and put an ON/OFF switch back on the circuit so it looks like the one shown.

2. Delete the bottom circuit so that only one full adder circuit remains.

3. CEDAR logic has 10 built-in pages that can be used to draw multiple parts of a circuit. These circuits can be wired together using TO/FROM gates.

4. Select the part of the circuit shown.

3. Copy the circuit and paste it onto Page 2.

 - To do this, click on the tab named Page 2

- Once you have clicked on the Page 2 tab, simply paste the circuit onto that page.

Notice how the inputs are still low because the TO/FROM gates go between pages.
