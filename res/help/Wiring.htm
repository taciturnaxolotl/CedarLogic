Wiring

CEDAR logic uses a Manhattan style wiring layout which means that wires can only go horizontal and vertical. Cedar logic implements an algorithm that combines Manhattan wiring with rubberbanding. This algorithm is called Mattwire. Mattwire gives the user good control over the circuit layout, but allows for some automated routing. Once a wire has been connected the user may select any vertical or horizontal section of the wire and drag it to a desired location.

1. This is the default wiring position.

2. To move the wire, simply place the mouse cursor the desired vertical or horizontal section of the wire that you wish to move so that a red x appears and left click.

3. Drag the section of the wire until it is in the desired position. The whole wire will be selected after you move a section of a wire.

4. Note that this can be done multiple times to any connection.
