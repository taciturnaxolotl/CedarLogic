ZAD System calls

The following ZAD system calls have been emulated and can be called in emulated ZAD code by calling to the appropriate addresses when the emulated Z80 chip is in On or Smart ZAD mode.

Name

Address

Description

KBRD

0x0FEB

Waits for a key press and returns a soft keycode.  This method also powers the hex display.

SCAN

0x0FEE

Scans the keyboard once and returns a hard keycode.  This method also powers the hex display.

HEXTO7

0x0FF1

Prepares the six hex digits in the DISPV for displaying on the hex display by KBRD or SCAN

MESOUT

0x0FF4

Prepares the six digit ASCII string pointed to by IX for displaying on the hex display by KBRD or SCAN

DELAY

0x0FFD

Pauses the system by the amount specified in the HL register pair.

DISPV

0x1F12

The start of a 6 byte vector used as an argument to HEXTO7

They keypad in the Z-80 Control panel can be accessed by the SCAN or KBRD method.  The SCAN method returns a Hard code while the KBRD returns a Soft code.
