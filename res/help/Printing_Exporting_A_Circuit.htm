Printing/Exporting A Circuit

1. To print a circuit, simply select print from the file menu. A standard Windows print dialog will appear.

 

2. To export a circuit, select Export bitmap from the edit menu. This will copy the current viewable portion of the circuit onto the clipboard. This function is good for copying a  picture of the circuit into a word-processor for doing a report or for copying a picture of the circuit schematic into a photo-editing program.
