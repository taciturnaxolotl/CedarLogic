ROM

Pin(s)

Function

Address

Supplies the ROM address from which the data will be outputted.

Output

Supples the output data from the current address of the ROM

OE (Output Enable)

When high, the data stored in the current address specified by the address pins will be placed on the Output pins. When low, the value will be high impedance.

Note: To load or save ROM files, double click on the ROM.
